## 練習問題
### 問題1

変数timesにリスト型で代入して画面に出力してください。

[出力結果]

```
[12.24, 13.02, 12.24, 12.89, 13.12]
```

### 問題2
問題1のタイムのうち、最速のものをfastest変数に代入し、画面に出力してください。

[出力結果]

```

```

### 問題3

[出力結果]

```

```
