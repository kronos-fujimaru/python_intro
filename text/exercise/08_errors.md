## 練習問題
### 問題1
以下の数値のリストを用意してください。

```
nums = [100, 200, 400, 0, 300]
```

以下のコードでエラーが発生する時、errorと出力するように修正してください。エラーが発生しない場合は、successと出力するように修正してください。

```python
print(nums[5])
```

[出力結果]

```
error
```

> `print(nums[4])`に変更して、300とsuccessが表示されることを確認しましょう。
